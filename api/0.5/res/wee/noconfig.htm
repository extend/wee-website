Creating a configuration file

You can create a configuration file by copying the sample file available at share/conf/sample.cnf. Copy it to app/conf/wee.cnf and modify it to suit your needs.

You can choose to use a different filename for your configuration file. In this case, please do not forget to define the constant WEE_CONF_FILE in your index.php file. This constant defines the path to your configuration file.
